# BBC News Scraper

This Python project scrapes the latest 10 BBC news articles using RSS feed and saves them into a CSV file.

## Features
- Fetches the latest 10 articles from BBC News RSS feed
- Saves title, link, summary, and publication date
- Outputs data to `bbc_news_scraped.csv`

## Requirements
- Python 3.x
- `requests` library
- `beautifulsoup4` library
- `pandas` library

## Usage
```bash
python bbc_news_scraper.py
